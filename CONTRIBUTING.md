# How to Contribute to SocioPredict

We're excited you want to help improve SocioPredict! Here's how to get started.

## Making Contributions

1. **Create Your Fork**
   - Fork this repository to your GitHub account first.

2. **Get the Code Locally**
   - Clone your fork to work on it.

3. **Work in a Branch**
   - Always create a new branch before making changes.
   - Name it descriptively: `feature/market-search` or `fix/login-redirect`.
   - Start branch names with `feature`, `fix`, `refactor`, or `doc`.

4. **Make Focused Commits**
   - Each commit should tackle one specific thing.
   - Write meaningful commit messages.

5. **Push Your Work**
   - Send your changes to your GitHub fork.

6. **Open a Pull Request**
   - Submit a PR to the main repository.
   - Explain what you changed and why.

## Found a Bug or Have an Idea?

Head to [Issues](https://github.com/YOUR_USERNAME/sociopredict/issues) to report bugs or suggest features. Before opening a new issue, check if someone already reported it. Include:

- A descriptive title
- Relevant details and context
- Steps to reproduce (for bugs)

## Community Standards

Please read our [Code of Conduct](https://github.com/YOUR_USERNAME/sociopredict/blob/main/CODE_OF_CONDUCT.md) before participating.

## Need Help?

Start a discussion on GitHub or open an issue.

Thanks for helping make SocioPredict better!
